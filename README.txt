My resume codebase


Directory Structure
-------------------
css\              - Stylesheets CSS files for the resume.
images\           - Images used in the resume.
scripts\          - JavaScript used in the resume.
styles\           - SCSS source files for stylesheets CSS. Available in Purchased versions only.
favicon.ico       - Favicon placeholder provided for the resume.
index.html        - Main HTML page to open the resume in browser.
*.html            - Additional HTML pages (if any).
